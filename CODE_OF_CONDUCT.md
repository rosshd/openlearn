# Code of Conduct

## Our Standard

openLearn should be a welcoming project for learners, educators, and developers.

Participants are expected to:

- Be respectful and constructive.
- Assume good intent, while taking responsibility for impact.
- Keep discussions focused on improving the project.
- Avoid harassment, personal attacks, and discriminatory language.
- Respect user privacy and never request private notes, API keys, or credentials in public discussions.

## Enforcement

Project maintainers may remove comments, close issues, or block participants who repeatedly act against these expectations.

If you need to report a conduct issue, contact the maintainer privately.
